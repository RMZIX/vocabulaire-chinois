📖 Grammaire

Règles de Grammaire

Cliquez sur un exemple pour l'entendre 🔊

Structure de base : Sujet + Verbe + Objet
Le chinois suit la même structure que le français : S → V → O. Il n'y a pas de conjugaison — le verbe ne change jamais de forme.
Sujet+Verbe+Objet
我 吃 饭
Wǒ chī fàn
Je mange 🔊
她 喝 茶
Tā hē chá
Elle boit du thé 🔊
我们 学习 中文
Wǒmen xuéxí zhōngwén
Nous étudions le chinois 🔊
他 看 书
Tā kàn shū
Il lit un livre 🔊
💡 Adjectifs : Ils se placent AVANT le nom. 好医生 = bon médecin. On utilise 很 (hěn = très) avec les adjectifs.
La Négation : 不 (bù) et 没 (méi)
Deux mots pour tout nier. Ils se placent toujours avant le verbe.
不 bù
Négation générale
✔ Refus / volonté
✔ Adjectifs
✔ Habitudes / Futur
没 méi
Négation passé / possession
✔ Action passée
✔ Possession (没有)
✔ Expérience vécue
我 不 吃 肉
Wǒ bù chī ròu
Je ne mange pas de viande 🔊
我 没 吃 饭
Wǒ méi chī fàn
Je n'ai pas mangé 🔊
我 没有 时间
Wǒ méiyǒu shíjiān
Je n'ai pas le temps 🔊
他 不是 医生
Tā bù shì yīshēng
Il n'est pas médecin 🔊
💡 Règle d'or : 是 → toujours nié avec 不是. 有 → toujours nié avec 没有.

Les Classificateurs (量词)
On ne peut pas mettre un nombre directement devant un nom. Il faut un classificateur entre les deux.
Nombre+量词+Nom
个 gè
Usage général
三个人sān gè rén — 3 personnes 🔊
本 běn
Livres
两本书liǎng běn shū — 2 livres 🔊
片 piàn
Comprimés / tranches
两片药liǎng piàn yào — 2 comprimés 🔊
杯 bēi
Tasses
一杯茶yī bēi chá — 1 tasse de thé 🔊
💡 两 vs 二 : Compter seul → 二. Devant un classificateur → 两 : 两个人, 两片药.
Exprimer le Temps
Le temps se place en début de phrase ou après le sujet. Le verbe ne change pas.
Temps+Sujet+Verbe+Objet
📅 Passé — 了 (le)

Wǒ chī le
J'ai mangé 🔊
昨天 我 学习
Zuótiān wǒ xuéxí
Hier j'ai étudié 🔊
⏳ En cours — 在 / 正在
我 在 学习
Wǒ zài xuéxí
Je suis en train d'étudier 🔊
她 正在 睡觉
Tā zhèngzài shuìjiào
Elle est en train de dormir 🔊
🔮 Futur — 要 / 明天
我 要 去 医院
Wǒ yào qù yīyuàn
Je vais à l'hôpital 🔊
明天 我们 去 上课
Míngtiān wǒmen qù shàngkè
Demain nous allons en cours 🔊
💡 Mots utiles : 现在 maintenant · 今天 aujourd'hui · 明天 demain · 昨天 hier · 每天 chaque jour

Les briques de base de l'écriture chinoise · Cliquez pour entendre 🔊

Pourquoi apprendre les radicaux ?
Les caractères chinois sont construits à partir de radicaux (部首 bùshǒu). Connaître ~30 radicaux vous permet de deviner le sens de centaines de nouveaux caractères. Par exemple, tous les caractères contenant 氵 (eau) ont souvent un lien avec l'eau : 海 mer · 河 rivière · 洗 laver · 游 nager.
